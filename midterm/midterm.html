A L G O R I T H M S

For various projects, I've had to create different kinds of algorithms. This ranges from color value algoritms on the image filters to regular algorithms for math-based projects. However, my Magnum-opus of algorithms comes in the form of a prisoner's dillema game theory algorithm. One school project we had was to make an algorithm that would be able to win a prisoner's dillema, were it to be pitted against another algorithm. We then compiled all the different strategies together and put them into a game theory battle arena where every person's code played the dillema against every other person's code 100 times each. Then, whichever person had the most points at the end officially came up with the best game theory strategy and put it into algorithm form. That year, I won. The following year, I asked my teacher to put me into that class's game as well, just for fun. I won that one too. This repeated for the next two years, essentially proving that I had created an unbeatable game theory algorithm! 4 years straight and counting (though that project wasn't done this year I heard due to covid).
